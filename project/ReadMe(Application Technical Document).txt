How my porject works:
In terminal window, type npm run dev, then the serve start run in localhost:5000

In terminal window,change folder to client:  cd client

In terminal window,type npm run serve to run client site.

In the browser, type url: localhost:8080.


The dependencies and components I used in my project are:
1. Cors in Javascript
CORS is a mechanism which aims to allow requests made on behalf of you and at the same time block some requests made by rogue JS 

2. Express
It's a web framework that let structure a web application to handle multiple different http requests at a specific url.

3.Body-parser 
It is the Node. js body parsing middleware. It is responsible for parsing the incoming request bodies in a middleware.

4.mongodb
nosql database

5. Axios
promise-based HTTP client that sports an easy-to-use API and can be used in both the browser and Node. js

6. Vue
It is an open-source progressive JavaScript framework for building user interfaces.
